Running the program

Running the program is easy. On your Unix machine, or the Cygwin prompt under Win/NT or Win/98, type:
  • dealer [FLAGS] < Input_file
This will start the program, read the Input_file and print the output on your screen. Alternatively, you can use:
  • dealer [FLAGS] < Input_file >Output_file
This will write the output to a file "Output_file".

The input_file argument can also be passed to the program in the normal way, instead of being used for input-redirection:

  • dealer [FLAGS] Input_file >Output_file
